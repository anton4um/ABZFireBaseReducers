General Requirements for the test task


Users want to fined answers to their questions quickly and data shows that people really care about how quickly thier pages load. The search team announced speed would be a ranking signal for desktop search in 2010 and as of this month (July 2018), page speed will be a ranking factor for mobile searches too.

If you`re a developer working on a site, now is a good time to evaluate youre performance using our speed tools. Think about how perfomance affects the user experience of youre page and consider measuring a variaty of real-world user-centric perfomance metric.

Are you shipping too much JavaScript? Too many images? Images and JavaScript are the most significant contributors to the page weight that affect page load time based on data from HTTP Achive and the Chrome User Experience Report - our public dataset for key UX metric as experienced by Chrome users under real-world conditions.
